# Project Demos

A collection of interactive demonstrations showcasing my experimental projects and utility libraries built with React and TypeScript.

## Overview

This application serves as a portfolio and testing ground for various UI components, hooks, and interactive features I've developed. Each demo is self-contained and demonstrates different approaches to common web development challenges.

## Projects

### useDrag

A React hook for adding drag functionality to HTML elements with optional boundary constraints.

**Key Features:**
- Drag-and-drop functionality for any HTML element
- Touch event support for mobile devices
- Boundary constraints to keep elements within containers
- Direction limiting (x-axis only, y-axis only, or both)
- Custom boundary checking logic
- Transform-based and object-position positioning
